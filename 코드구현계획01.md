```python
@dataclass
class CufsmModel:
    prop: np.ndarray      # (nmats, 6) [matnum, Ex, Ey, vx, vy, G]
    node: np.ndarray      # (nnodes, 8) [node#, x, z, dofx, dofz, dofy, dofrot, stress]
    elem: np.ndarray      # (nelems, 5) [elem#, nodei, nodej, t, matnum]
    lengths: np.ndarray   # (nlengths,)
    springs: np.ndarray   # (nsprings, 10) or empty
    constraints: np.ndarray  # (nconstraints, 5) or empty
    BC: str               # 'S-S', 'C-C', 'S-C', 'C-F', 'C-G'
    m_all: list           # list of np.ndarray (종방향 항)

    glob: np.ndarray
    dist: np.ndarray
    local: np.ndarray
    other: np.ndarray
    ospace: int = 1
    orth: int = 1
    couple: int = 1
    norm: int = 0
```

### Step 1-3: 요소 강성행렬 (element.py)

> **참조:** 프로젝트개요.md §5.1 — klocal.m, kglocal.m 설명
> **참조:** 컨버전전략.md §6 — eigs, sparse 매핑

**파일:** `python/engine/element.py`

**원본 MATLAB:**
- `klocal(Ex,Ey,vx,vy,G,t,a,b,BC,m_a)` → `k` (8·totalm × 8·totalm)
- `kglocal(a,b,Ty1,Ty2,BC,m_a)` → `kg` (8·totalm × 8·totalm)

**Python 구현:**
```python
def klocal(Ex, Ey, vx, vy, G, t, a, b, BC, m_a) -> np.ndarray:
    """요소 탄성 강성행렬. 반환: (8*totalm, 8*totalm) dense array"""

def kglocal(a, b, Ty1, Ty2, BC, m_a) -> np.ndarray:
    """요소 기하강성행렬. 반환: (8*totalm, 8*totalm) dense array"""

def spring_klocal(ku, kv, kw, kq, a, BC, m_a, discrete, ys) -> np.ndarray:
    """스프링 요소 강성행렬"""
```

**핵심 로직:**
1. 판 강도(rigidity) 계산: `E1 = Ex/(1-vx*vy)`, `D1 = Ex*t³/(12*(1-vx*vy))` 등
2. 이중 루프 `(m, p)` over `m_a` → `BC_I1_5(BC, m_a[m], m_a[p], a)` 호출
3. 4×4 멤브레인 블록 `km_mp` + 4×4 휨 블록 `kf_mp` → 8×8 블록
4. 전체 행렬 `k[8*m:8*(m+1), 8*p:8*(p+1)] = [[km_mp, 0], [0, kf_mp]]`

**검증:** 단일 요소에 대해 MATLAB 출력과 행렬 원소 비교 (상대오차 < 1e-10)

### Step 1-4: 경계조건 적분 (boundary.py)

> **참조:** 프로젝트개요.md §6 경계조건 지원 (5가지 BC)
> **참조:** 원본 `Ref_Source/analysis/BC_I1_5.m`

**파일:** `python/engine/boundary.py`

**원본 MATLAB:**
- `BC_I1_5(BC, kk, nn, a)` → `[I1, I2, I3, I4, I5]`

**적분 정의:**
- `I1 = ∫₀ᵃ Yₘ·Yₙ dx` (형상함수 곱)
- `I2 = ∫₀ᵃ Yₘ″·Yₙ dx`
- `I3 = ∫₀ᵃ Yₘ·Yₙ″ dx`
- `I4 = ∫₀ᵃ Yₘ″·Yₙ″ dx`
- `I5 = ∫₀ᵃ Yₘ′·Yₙ′ dx`

**BC별 형상함수:**
- `S-S`: `Yₘ = sin(mπx/a)` — m≠n이면 모두 0 (직교)
- `C-C`: 복합 삼각함수, `|m-n|≤2`일 때만 비영
- `S-C`: 혼합, `|m-n|≤1`일 때만 비영
- `C-F`: 비대칭 공식
- `C-G`: 부분 구속 공식

**구현:** 닫힌 형태(closed-form) 수식 직접 번역 — 수치 적분 불필요

**추가 함수:**
```python
def BC_I1_5_atpoint(BC, kk, nn, a, ys) -> tuple:
    """이산 스프링용 — 특정 위치 ys에서의 I1, I5만 반환"""
```

### Step 1-6: 전체 행렬 조립 (assembly.py)

> **참조:** 프로젝트개요.md §4 해석 워크플로우 — 전체 행렬 조립 단계
> **참조:** 원본 `assemble.m`, `spring_assemble.m`

**파일:** `python/engine/assembly.py`

```python
def assemble(K, Kg, k, kg, nodei, nodej, nnodes, m_a) -> tuple:
    """요소 행렬을 전체 행렬에 조립
    K, Kg: scipy.sparse.lil_matrix (4*nnodes*totalm × 4*nnodes*totalm)
    k, kg: 요소 행렬 (8*totalm × 8*totalm)
    반환: (K, Kg) 업데이트된 전체 행렬"""

def spring_assemble(K, k_spring, nodei, nodej, nnodes, m_a) -> sparse:
    """스프링 강성을 전체 행렬에 추가 (nodej=0: 접지 스프링)"""
```

**DOF 배치 규칙 (MATLAB → Python):**
```
MATLAB:  K(4*nnodes*(i-1) + nodei*2-1, ...)  (1-based)
Python:  K[4*nnodes*i + (nodei-1)*2, ...]     (0-based, nodei는 원본 1-based 값에서 -1)
```

- 멤브레인 DOF (u,v): 처음 2·nnodes 행
- 휨 DOF (w,θ): 다음 2·nnodes 행
- m항마다 4·nnodes 블록 반복

**핵심:** `scipy.sparse.lil_matrix`로 조립 → 풀이 전 `.tocsr()` 변환

### Step 1-7: FSM 솔버 (fsm_solver.py)

> **참조:** 프로젝트개요.md §4 해석 워크플로우 전체, §5.1 stripmain.m 설명
> **참조:** 컨버전전략.md §6 — `eigs` → `eigsh` 매핑

**파일:** `python/engine/fsm_solver.py`

```python
def stripmain(prop, node, elem, lengths, springs, constraints,
              GBTcon, BC, m_all, neigs=20) -> CufsmResult:
    """FSM 좌굴 해석 메인 루틴

    프로젝트개요.md §4 워크플로우를 그대로 구현:
    [1] elemprop() → 요소 폭, 각도
    [2] 각 길이(length)에 대해:
        a. K, Kg 초기화 (sparse zero)
        b. 각 요소에 대해:
           - klocal() → k (로컬 강성)
           - kglocal() → kg (로컬 기하강성)
           - trans() → k_global, kg_global (좌표변환)
           - assemble() → K, Kg (전체 조립)
        c. 스프링 조립 (있을 경우)
        d. 구속조건 적용 (user constraints → null space reduction)
        e. 경계조건 적용 (DOF 제거)
        f. 고유치 풀이: eigsh(K, M=Kg, k=neigs, sigma=0)
        g. 양의 실수 고유치 추출 → 좌굴 하중비
        h. 모드형상 정규화 (최대 절대값 = 1)
    [3] curve, shapes 반환
    """
```

**고유치 풀이 핵심:**
```python
from scipy.sparse.linalg import eigsh

# Kg 대칭화
Kg_sym = (Kgff + Kgff.T) / 2.0
# shift-invert 모드로 최소 고유치 탐색
eigenvalues, eigenvectors = eigsh(Kff, M=Kg_sym, k=min(2*neigs, ndof-1), sigma=0, which='LM')
```

**구속조건 축소:**
```python
from scipy.linalg import null_space
R = null_space(R0.T)
Kff = R.T @ K_reduced @ R
Kgff = R.T @ Kg_reduced @ R
```

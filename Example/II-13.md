## 6. DSM Calculation Results

### 6a. Yielding and Global Buckling (Section F2)

Mne = My = 86.4 kip-in. (fully braced)

### 6b. Local Buckling (Section F3)

| Parameter | Symbol | Value |
|-----------|--------|-------|
| Slenderness | lambda_l | sqrt(86.4/82.9) = 1.02 |
| Nominal strength | Mnl | 72.4 kip-in. |

Eq. F3.2.1-2: Mnl = [1 - 0.15*(Mcrl/Mne)^0.4] * (Mcrl/Mne)^0.4 * Mne

### 6c. Distortional Buckling (Section F4)

| Parameter | Symbol | Value |
|-----------|--------|-------|
| Slenderness | lambda_d | sqrt(86.4/100) = 0.930 |
| Nominal strength | Mnd | 71.0 kip-in. |

Eq. F4.1-2: Mnd = [1 - 0.22*(Mcrd/My)^0.5] * (Mcrd/My)^0.5 * My

### 6d. Controlling Limit State

**Distortional buckling controls** (Mnd = 71.0 kip-in. < Mnl = 72.4 kip-in.)

### 6e. Available Strengths

| Method | Factor | Design Strength | Unit |
|--------|--------|----------------|------|
| ASD | Omega = 1.67 | Ma = 71.0/1.67 = 42.5 | kip-in. |
| LRFD | phi = 0.90 | phi*Mn = 0.90*71.0 = 63.9 | kip-in. |

## 7. Key Features for Validation

- **Complex section**: Sigma section with web stiffener - EWM has no provisions for this
- **DSM well-suited**: Demonstrates DSM advantage for non-standard sections
- **Buckling curve graph**: Shows Mcr/My vs half-wavelength with mode shapes
- **Close competition**: Local (72.4) and distortional (71.0) nearly equal - distortional barely controls
- **Complete design**: Full ASD and LRFD available strengths provided

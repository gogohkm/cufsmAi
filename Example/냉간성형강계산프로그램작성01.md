# 냉간성형강 설계 계산 프로그램 확장 계획서

## cufsmAi + AISI S100-16 통합 설계 시스템

---

## 1. 개요

### 1.1 목적
cufsmAi의 기존 프리프로세서/해석/포스트프로세서 인프라를 그대로 유지하면서,
AISI S100-16 기반 냉간성형강 **설계(Design)** 기능을 추가한다.

### 1.2 핵심 설계 원칙
1. **기존 탭 보존**: Preprocessor / Analysis / Postprocessor 탭은 변경 없음
2. **새 탭 추가**: `Design` 탭을 4번째 탭으로 추가
3. **AI + 사람 동시 작업**: MCP 도구를 통한 AI 주도 설계와 UI를 통한 사람 주도 설계를 동시 지원
4. **CUFSM 연계**: 좌굴 해석 결과(Pcrl, Pcrd, Mcrl, Mcrd)를 설계 계산에 자동 연계

### 1.3 지원 범위 (74개 예제 기준)

| 카테고리 | 예제 수 | 우선순위 | 설명 |
|---------|---------|---------|------|
| Part I: 단면 성질 | 20 | ★★★ | 총단면/유효단면 성질 계산 |
| Part II: 보 설계 | 20 | ★★★ | 휨강도 (EWM/DSM) |
| Part III: 기둥 설계 | 19 | ★★★ | 압축강도 (EWM/DSM) |
| Part IV: 접합부 설계 | 12 | ★★ | 용접/볼트/나사 접합 |
| Part V: 보충 | 1 | ★ | 폰딩 해석 |
| Part VI: 시험 절차 | 2 | ★ | 시험 데이터 보정 |

## 2. 현재 시스템 분석

### 2.1 기존 MCP 도구 (재사용 가능)

현재 cufsmAi에 이미 구현된 MCP 도구 중 설계 계산에 **직접 활용 가능**한 것들:

| MCP 도구 | 기능 | 설계 활용 |
|---------|------|----------|
| `set_section_template` | 단면 생성 (C, Z, Hat 등) | 설계 대상 단면 정의 |
| `get_section_properties` | A, Ixx, Izz, J, Cw 등 | 총단면 성질 → 설계 입력 |
| `get_cutwp` | 전단중심, 뒤틀림 상수 | 비틀림 좌굴 계산 |
| `run_analysis` | FSM 좌굴 해석 | Fcrl, Fcrd, Fcre 추출 |
| `get_dsm_values` | Pcrl, Pcrd, Mcrl, Mcrd, Py, My | DSM 설계값 → 설계 입력 |
| `get_buckling_curve` | 반파장-하중계수 곡선 | 좌굴 모드 식별 |
| `classify_modes` | G/D/L/O 모드 분류 | 지배 좌굴 모드 판별 |
| `set_material` | E, v, G 설정 | 재료 성질 정의 |
| `set_stress` | 응력 분포 설정 | 하중 케이스 정의 |
| `run_plastic_surface` | P-M 소성 상호작용곡면 | 소성 강도 확인 |

### 3.2 Member Type별 입력 필드

#### 3.2.1 Beam (휨 부재)
```
Required:
  - 단면 (Preprocessor에서 정의됨)
  - Fy, Fu (재료)
  - Lb (횡지지 간격, in)
  - Cb (모멘트 구배 계수, 기본값 1.0)
  - Mu 또는 Ma (소요 휨 모멘트)
  
From CUFSM (자동):
  - Mcrl (국부 좌굴 임계 모멘트)
  - Mcrd (왜곡 좌굴 임계 모멘트)
  - My = Sf × Fy (항복 모멘트)
```

#### 3.2.2 Column (압축 부재)
```
Required:
  - 단면 (Preprocessor에서 정의됨)
  - Fy, Fu (재료)
  - KxLx, KyLy, KtLt (유효 좌굴 길이)
  - Pu 또는 Pa (소요 축력)

From CUFSM (자동):
  - Pcrl (국부 좌굴 임계 하중)
  - Pcrd (왜곡 좌굴 임계 하중)
  - Py = Ag × Fy (항복 하중)
```

#### 3.2.3 Beam-Column (조합 하중)
```
Required:
  - Beam + Column 입력 모두
  - Cmx, Cmy (등가 모멘트 계수)
  
Output:
  - 개별 강도 (Pn, Mnx, Mny)
  - 상호작용 비율: P/Pa + Mx/Max + My/May ≤ 1.0
```

#### 3.2.4 Connection (접합)
```
Required:
  - 접합 유형 (용접/볼트/나사/PAF)
  - 접합 상세 (크기, 개수, 간격)
  - 모재 두께, 강도
  
Output:
  - 각 파괴 모드별 강도
  - 지배 파괴 모드
  - 설계 강도
```

### 4.2 도구 활용 시나리오

#### 시나리오 A: AI 주도 설계 (대화창)

```
사용자: "9CS2.5x059 C형강을 사용해서 퍼린 설계를 해줘. 
        Fy=55ksi, 횡지지 간격 56.3인치, DSM으로."

AI 워크플로우:
  1. aisi_design_guide(query_type="beam", section_type="C", method="DSM")
     → 필요 입력/절차/공식 확인
     
  2. set_section_template(type="lippedc", H=9, B=2.5, t=0.059)
     → 단면 생성
     
  3. set_material(E=29500, v=0.3)
     → 재료 설정
     
  4. set_stress(type="pure_bending", fy=55)
     → 휨 응력 분포 설정
     
  5. run_analysis(neigs=10)
     → FSM 좌굴 해석 실행
     
  6. get_dsm_values(fy=55)
     → Mcrl, Mcrd, My 추출
     
  7. aisi_design_flexure(method="DSM", design_method="LRFD",
                         Fy=55, Fu=70, Lb=56.3, Cb=1.67)
     → 설계 계산 실행
     
  8. 결과 보고: Mn=84.2 kip-in, 국부좌굴 지배, φMn=75.8 kip-in
```

#### 5.2.1 DSM 강도 곡선 (`dsm_strength.py`)

```python
def dsm_compression_local(Pne: float, Pcrl: float) -> dict:
    """DSM 국부좌굴 압축강도 (AISI S100 §E3.2.1)
    
    Args:
        Pne: 전체좌굴 공칭강도 (kips)
        Pcrl: 탄성 국부좌굴 임계하중 (kips)
    
    Returns:
        dict: {Pnl, lambda_l, equation_used}
    """
    lambda_l = math.sqrt(Pne / Pcrl)
    
    if lambda_l <= 0.776:
        Pnl = Pne                           # Eq. E3.2.1-1
        eq = "E3.2.1-1"
    else:
        ratio = (Pcrl / Pne) ** 0.4
        Pnl = (1 - 0.15 * ratio) * ratio * Pne  # Eq. E3.2.1-2
        eq = "E3.2.1-2"
    
    return {'Pnl': Pnl, 'lambda_l': lambda_l, 'equation': eq}


def dsm_compression_distortional(Py: float, Pcrd: float) -> dict:
    """DSM 왜곡좌굴 압축강도 (AISI S100 §E4.1)
    
    Args:
        Py: 항복하중 = Ag × Fy (kips)
        Pcrd: 탄성 왜곡좌굴 임계하중 (kips)
    
    Returns:
        dict: {Pnd, lambda_d, equation_used}
    """
    lambda_d = math.sqrt(Py / Pcrd)
    
    if lambda_d <= 0.561:
        Pnd = Py                             # Eq. E4.1-1
        eq = "E4.1-1"
    else:
        ratio = (Pcrd / Py) ** 0.6
        Pnd = (1 - 0.25 * ratio) * ratio * Py   # Eq. E4.1-2
        eq = "E4.1-2"
    
    return {'Pnd': Pnd, 'lambda_d': lambda_d, 'equation': eq}


def dsm_flexure_local(Mne: float, Mcrl: float) -> dict:
    """DSM 국부좌굴 휨강도 (AISI S100 §F3.2.1)"""
    lambda_l = math.sqrt(Mne / Mcrl)
    
    if lambda_l <= 0.776:
        Mnl = Mne                            # Eq. F3.2.1-1
        eq = "F3.2.1-1"
    else:
        ratio = (Mcrl / Mne) ** 0.4
        Mnl = (1 - 0.15 * ratio) * ratio * Mne  # Eq. F3.2.1-2
        eq = "F3.2.1-2"
    
    return {'Mnl': Mnl, 'lambda_l': lambda_l, 'equation': eq}


def dsm_flexure_distortional(My: float, Mcrd: float) -> dict:
    """DSM 왜곡좌굴 휨강도 (AISI S100 §F4.1)"""
    lambda_d = math.sqrt(My / Mcrd)
    
    if lambda_d <= 0.673:
        Mnd = My                             # Eq. F4.1-1
        eq = "F4.1-1"
    else:
        ratio = (Mcrd / My) ** 0.5
        Mnd = (1 - 0.22 * ratio) * ratio * My   # Eq. F4.1-2
        eq = "F4.1-2"
    
    return {'Mnd': Mnd, 'lambda_d': lambda_d, 'equation': eq}
```

#### 5.2.2 전체 좌굴 (`global_buckling.py`)

```python
def flexural_buckling(E, Fy, KL, r):
    """유연좌굴 임계응력 (§E2)
    
    Returns: {Fcre, lambda_c, Fn, equation}
    """
    Fcre = math.pi**2 * E / (KL / r)**2
    lambda_c = math.sqrt(Fy / Fcre)
    
    if lambda_c <= 1.5:
        Fn = (0.658 ** (lambda_c**2)) * Fy     # Eq. E2-2
        eq = "E2-2"
    else:
        Fn = (0.877 / lambda_c**2) * Fy         # Eq. E2-3
        eq = "E2-3"
    
    return {'Fcre': Fcre, 'lambda_c': lambda_c, 'Fn': Fn, 'equation': eq}


def torsional_buckling(E, G, Ag, J, Cw, ro, KtLt):
    """비틀림좌굴 임계응력 (§E2.1)"""
    sigma_t = (1 / (Ag * ro**2)) * (G * J + math.pi**2 * E * Cw / KtLt**2)
    return sigma_t


def flexural_torsional_buckling(sigma_ex, sigma_t, beta):
    """휨-비틀림좌굴 임계응력 (§E2.2, 단축대칭 단면)"""
    Fcre = (1 / (2*beta)) * ((sigma_ex + sigma_t) 
            - math.sqrt((sigma_ex + sigma_t)**2 - 4*beta*sigma_ex*sigma_t))
    return Fcre


def lateral_torsional_buckling(Cb, ro, Ag, Sf, sigma_ey, sigma_t):
    """횡-비틀림좌굴 임계응력 (§F2.1)"""
    Fcre = (Cb * ro * Ag / Sf) * math.sqrt(sigma_ey * sigma_t)
    return Fcre
```

#### 5.2.3 상호작용 검토 (`interaction.py`)

```python
def combined_axial_bending(P, Pa, Mx, Max, My=0, May=1):
    """조합하중 상호작용 검토 (§H1.2, Eq. H1.2-1)
    
    Returns: {P_ratio, Mx_ratio, My_ratio, total, pass}
    """
    P_ratio = P / Pa if Pa > 0 else 0
    Mx_ratio = abs(Mx) / Max if Max > 0 else 0
    My_ratio = abs(My) / May if May > 0 else 0
    total = P_ratio + Mx_ratio + My_ratio
    
    return {
        'P_ratio': P_ratio,
        'Mx_ratio': Mx_ratio,
        'My_ratio': My_ratio,
        'total': total,
        'pass': total <= 1.0,
        'equation': 'H1.2-1',
    }


def combined_bending_shear(M, Mao, V, Va):
    """휨-전단 상호작용 검토 (§H2, Eq. H2-1)"""
    M_ratio = (M / Mao) ** 2
    V_ratio = (V / Va) ** 2
    total = math.sqrt(M_ratio + V_ratio)
    
    return {
        'M_ratio': math.sqrt(M_ratio),
        'V_ratio': math.sqrt(V_ratio),
        'total': total,
        'pass': total <= 1.0,
        'equation': 'H2-1',
    }
```

## 15. Required Loads (소요 ��중) 계산 시스템 — 예제 분석 및 구현 계획

### 15.1 개요 및 동기

현재 Design 탭의 "Applied Loads" 입력(P, Mx, My, V)은 사용자가 **직접 소요하중값을 입력**하도록 되어 있다. 그러나 실제 설계에서는:

1. **하중 정의**: 고정하중(D), 활하중(L/Lr), 설하중(S), 풍하중(W) 등 개별 하중 정의
2. **하중 조합**: ASCE/SEI 7 기준에 따라 ASD 또는 LRFD 하중 조합 자동 적용
3. **구조해석**: 부재 길이, 경계조건, 연속성에 따른 모멘트/전단/반력 자동 계산
4. **임계값 추출**: 정/부 모멘트 영역별 최대값, 최대 전단력, 지점 반력 추출

이 4단계가 **자동화**되어야 사용자가 하중만 정의하면 소요강도(Mu, Vu, Pu)를 자동으로 구할 수 있다.

### 15.2 AISI 예제에서 추출한 설계 패턴

#### 15.2.1 예제 II-1A/1B/1C: 4경간 연속 C-퍼린 (가장 포괄적)

**Given (주어진 조건):**
- 4경간 연속보, 각 스팬 25.0 ft
- C-퍼린: 외측 9CS2.5x070, 내측 9CS2.5x059
- Dead Load = 15 PLF, Roof Live Load = 90 PLF (1C) 또는 100 PLF (2A), Wind Uplift = 144~170 PLF
- 내부 지점에서 겹침 연결(laps): 외측 스팬 1.25ft + 2.75ft, 내측 스팬 유사
- 지붕 덮개가 상부 플랜지에 through-fastened (12 in. o.c.)
- Fy = 55 ksi

**설계 검토 체크리스트 (Section별):**

| 단계 | 검토 항목 | 하중 조합 | AISI 조항 | 비고 |
|------|----------|----------|----------|------|
| 1 | 구조해석 | 각 하중 개별 | — | 비등단면 연속보 해석, ΣIx 사용 |
| 2 | 중력 휨 검토 | 1.2D+1.6Lr (LRFD) 또는 D+Lr (ASD) | §F2, §F3, §F4, §I6.2.1 | 정/부 모멘트 영역별 |
| 3 | 중력 전단 검토 | 1.2D+1.6Lr 또는 D+Lr | §G2.1, §G2.3 | 지점 및 랩 끝 |
| 4 | 휨+전단 상호작용 | 동일 | §H2 | √((M/Ma)²+(V/Va)²) ≤ 1.0 |
| 5 | 웹 크리플링 | 1.2D+1.6Lr 또는 D+Lr | §G5 | 단부/내부 지점 구분 |
| 6 | 휨+웹크리플링 | 동일 | §H3 | 0.91(P/Pn)+(M/Mn) ≤ 1.33φ |
| 7 | 풍압 양력 검토 | 0.9D+1.0W (LRFD) 또는 0.6D+0.6W (ASD) | §I6.2.1 | R 감소계수 적용 |

**핵심 포인트 — 소요강도 계산:**

```
LRFD 중력 (지배 조합: 1.2D + 1.6Lr):
  Mu = 1.2×M_D + 1.6×M_Lr

  외측 스팬:
    최대 정모멘트: Mu = (1.2)(0.68) + (1.6)(4.09) = 7.36 kip-ft
    랩 끝 부모멘트: Mu = (1.2)(0.68) + (1.6)(4.08) = 7.34 kip-ft
    지점 부모멘트: Mu = (1.2)(1.11) + (1.6)(6.69) = 12.0 kip-ft

  내측 스팬:
    랩 끝 부모멘트: Mu = (1.2)(0.61) + (1.6)(3.63) = 6.54 kip-ft
    최대 정모멘트: Mu = (1.2)(0.29) + (1.6)(1.76) = 3.16 kip-ft
    중앙 지점 부모멘트: Mu = (1.2)(0.66) + (1.6)(3.98) = 7.16 kip-ft

LRFD 양력 (지배 조합: 0.9D + 1.0W):
  Mu = 0.9×M_D + 1.0×M_W

  외측 스팬:
    최대 부모멘트: Mu = (0.9)(0.68) + (1.0)(-6.54) = -5.93 kip-ft
```

```python
def get_controlling_combinations(member_type, loads, method='LRFD'):
    """
    부재 유형과 입력 하중에 따라 관련 하중 조합만 자동 선별
    
    member_type: 'roof_purlin', 'floor_joist', 'wall_stud', 'general_beam'
    loads: dict with keys 'D', 'L', 'Lr', 'S', 'W', 'Wu', 'E', 'R'
    method: 'ASD' or 'LRFD'
    
    Returns: list of (combination_name, factor_dict) 
    """
    combos = []
    
    if method == 'LRFD':
        # 항상 포함
        combos.append(('1.4D', {'D': 1.4}))
        
        # 활하중 있으면
        if loads.get('L'):
            combos.append(('1.2D+1.6L+0.5Lr', {'D': 1.2, 'L': 1.6, 'Lr': 0.5}))
        
        # 지붕 활하중 또는 설하중 있으면
        if loads.get('Lr') or loads.get('S'):
            Lr_or_S = max(loads.get('Lr', 0), loads.get('S', 0))
            combos.append(('1.2D+1.6Lr', {'D': 1.2, 'Lr': 1.6}))
            if loads.get('S'):
                combos.append(('1.2D+1.6S', {'D': 1.2, 'S': 1.6}))
        
        # 풍하중 있으면
        if loads.get('W') or loads.get('Wu'):
            combos.append(('1.2D+1.0W+L+0.5Lr', {'D': 1.2, 'W': 1.0, 'L': 1.0, 'Lr': 0.5}))
            combos.append(('0.9D+1.0W', {'D': 0.9, 'W': 1.0}))  # 양력!
        
        # 지진하중 있으면
        if loads.get('E'):
            combos.append(('1.2D+1.0E+L+0.2S', {'D': 1.2, 'E': 1.0, 'L': 1.0, 'S': 0.2}))
            combos.append(('0.9D+1.0E', {'D': 0.9, 'E': 1.0}))
    
    elif method == 'ASD':
        combos.append(('D', {'D': 1.0}))
        if loads.get('L'):
            combos.append(('D+L', {'D': 1.0, 'L': 1.0}))
        if loads.get('Lr') or loads.get('S'):
            combos.append(('D+Lr', {'D': 1.0, 'Lr': 1.0}))
            if loads.get('S'):
                combos.append(('D+S', {'D': 1.0, 'S': 1.0}))
        if loads.get('W') or loads.get('Wu'):
            combos.append(('D+0.6W', {'D': 1.0, 'W': 0.6}))
            combos.append(('0.6D+0.6W', {'D': 0.6, 'W': 0.6}))  # 양력!
    
    return combos
```

### 15.6 특수 고려사항

#### 15.6.1 비틀림 하중 (Torsional Loading, §H4, §I6.1.3)

C-단면/Z-단면에 하중이 전단중심(Shear Center)에 작용하지 않으면 비틀림이 발생한다.

```
비틀림 편심 e = m + t/2 + R   (m = 전단중심~웹 중심 거리)
비틀림 모멘트 tu = wu × e     (단위 길이당)

예: e = 1.05 + 0.070/2 + 0.1875 = 1.27 in.
    tu = 162 plf × 1.27 in. = 206 lb-in./ft
```

**구현에 필요한 입력:**
- 전단중심 좌표 (get_cutwp MCP에서 제공)
- 하중 작용점 (플랜지 상면 가정)
- 보강(brace) 조건 및 간격

#### 15.6.2 앵커리지력 (Anchorage Forces)

Z-단면 퍼린의 경우, 비대칭 단면에 의해 횡방향 변위 경향이 발생한다.
AISI D111-18 (Design Guide for Cold-Formed Steel Purlin Roof Framing Systems) 기준으로 검토.

```
PL = 0.5 × [C2/1000 × (Ixy×L)/(Ix×d) + C3 × (0.25×b×t)/d²] × (pts+pd) × s
  C2 = 8.3, C3 = 28 (Z-단면 상수)
  Ixy = 비대칭 단면2차모멘트
```

#### 15.6.3 양력 시 강도 감소계수 R

| 지붕 유형 | R 값 | 적용 조항 | 조건 |
|----------|------|----------|------|
| Through-fastened (C-퍼린) | 0.60 | §I6.2.1 | 15개 조건 충족 |
| Through-fastened (Z-퍼린) | 0.70 | §I6.2.1 | 15개 조건 충족 |
| Standing-seam | 시험 결정 | §I6.2.2 | AISI S908 시험 |

양력 시: `Mn = R × Mnℓ = R × Se × Fy`

#### 15.6.4 모멘트 구배 보정계수

**Cb (횡-비틀림 좌굴 보정):**
```
Cb = 12.5 Mmax / (2.5 Mmax + 3 MA + 4 MB + 3 MC)
  MA, MB, MC = 비지지 구간의 1/4, 1/2, 3/4 점 모멘트
```
예제에서는 보수적으로 Cb = 1.67 (선형 모멘트 가정) 사용

**β (왜곡 좌굴 구배 보정):**
```
β = 1 + 0.4(L/Lm)^0.7 × (1 + M1/M2)^0.7,  1.0 ≤ β ≤ 1.3
  L = min(Lcrd, Lm)
  M1 = 구간 시작점 모멘트 (보통 0, 변곡점)
  M2 = 구간 끝점 모멘트
```

#### 15.6.5 덱/패널 강성 기여

| 강성 유형 | 기호 | 설명 | 기여 항목 |
|----------|------|------|----------|
| 횡(전단) 강성 | kx | 패널의 횡변위 구속 | LTB 방지 (정모멘트) |
| 회전 강성 | kφ | 패널의 플랜지 회전 구속 | 왜곡 좌굴 저항 |
| 합성 강성 | kxo | 패널과 퍼린의 합성 효과 | 추가 강성 |

```
Fcrd = β × (k_φfc + k_φwc + k_φ) / (k_φfg + k_φwg)
  k_φ = 외부 회전 강성 (패널 등)
  k_φfc, k_φwc = 플랜지/웹 기여 (단면 고유값)
```

### 16.1 패널 회전강성 kφ 결정 방법 (★ 가장 중요한 미해결 입력)

현재 예제에서 kφ는 "주어진 값"으로만 표시되어 있으나, Example II-1C에서 **실제 계산 과정**을 상세히 보여준다.

#### 16.1.1 kφ 결정의 3가지 경로

| 경로 | 방법 | 정밀도 | 구현 가능성 |
|------|------|--------|-----------|
| **A. 시험** | AISI S901-17 시험 표준 | 가장 정확 | 사용자가 시험값 직접 입력 |
| **B. 연구기반 계산** | Chen & Moen (2011) 공식 | 중간 | ★ 코드로 구현 가능 |
| **C. 보수적 가정** | kφ = 0 (무지지) | 가장 보수적 | 기본값으로 사용 |

#### 16.1.2 경로 B: 패널 회전강성 계산 공식 (Example II-1C 기반)

**Step 1 — 패널 인발 강성 (per fastener)**

연구보고서(Chen & Moen 2011)에 따르면, FEM 시뮬레이션 결과 패널 두께 0.018 in.일 때 개별 패스너당 비틀림 강성은:

```
k_per_fastener = 1.77 kip/in./fastener  (t_panel ≥ 0.018 in. 기준)
```

**주의:** 패널 두께가 0.018 in. 미만이거나, 패스너가 위에서 아래로 체결된 경우 과대평가 가능.

**Step 2 — 연속 회전강성으로 변환**

패스너 간격 s (in. o.c.)로 나누어 단위길이당 강성으로 변환:

```
k = k_per_fastener / s = 1.77 / 12 = 0.1475 kip/in./in.
```

**Step 3 — 패스너~플랜지-웹 접합부 거리 고려**

패스너 위치가 플랜지 중앙에 있을 때, 플랜지-웹 접합부까지의 거리:

```
c = b/2 = 플랜지폭/2 - 웹두께(무시)
```

예: c = 2.5/2 = 1.25 in. (패스너가 플랜지 중앙에 위치)

**Step 4 — 최종 회전강성 계산**

```
kφ = 1 / (1/(k×c²) + c³/(3×E×I_panel×c²))

여기서:
  k = 패스너 인발 강성 / 간격 (kip/in./in.)
  c = 패스너~플랜지-웹 접합부 거리 (in.)
  E = 29500 ksi
  I_panel = t_purlin³/12 (in.⁴/in.)  ← 얇은 부재 두께 사용 (보수적)
```

**Example II-1C 계산 결과:**
```python
k = 1.77 / 12  # = 0.1475 kip/in./in.
c = 1.25        # in.
E = 29500       # ksi
t_min = 0.059   # in. (보수적으로 얇은 두께 사용)
I = t_min**3 / 12  # = 0.0000171 in.⁴/in.

kphi = 1 / (1/(k * c**2) + c**3 / (3 * E * I * c**2))
# kphi ≈ 0.113 kip-in./rad/in.
```

#### 16.1.3 코드 구현 — kφ 자동 계산

```python
def calc_rotational_stiffness_from_deck(
    t_panel: float,       # 패널(데크) 설계두께 (in.)
    t_purlin: float,      # 퍼린 두께 (in.) — 얇은 쪽
    fastener_spacing: float,  # 패스너 간격 (in.), 보통 12
    flange_width: float,  # 플랜지 폭 (in.)
    k_per_fastener: float = 1.77,  # kip/in./fastener (기본값)
    E: float = 29500.0    # ksi
) -> float:
    """
    패널-퍼린 연결의 회전강성 kφ 계산 (Chen & Moen 2011)
    
    Returns: kφ (kip-in./rad/in.)
    """
    # 패스너~웹 접합부 거리
    c = flange_width / 2.0  # 패스너가 플랜지 중앙에 위치한다고 가정
    
    # 단위 길이당 인발 강성
    k = k_per_fastener / fastener_spacing
    
    # 패널 관성모멘트 (단위폭당)
    t_min = min(t_panel, t_purlin)
    I_panel = t_min**3 / 12.0
    
    # 회전강성 (직렬 스프링 모델)
    kphi = 1.0 / (1.0 / (k * c**2) + c**3 / (3.0 * E * I_panel * c**2))
    
    return kphi  # kip-in./rad/in.
```

### 16.2 횡강성 kx 결정 방법

#### 16.2.1 계산 공식 (Example II-1C, AISI RP17-2 기반)

**Step 1 — 2개 플라이의 축강성 (직렬 스프링)**
```
kx_plate = [1/(E×t1) + 1/(E×t2)]⁻¹
  t1 = 패널 두께 (in.)
  t2 = 퍼린 두께 (in.)
  
예: kx_plate = [1/(29500×0.0179) + 1/(29500×0.059)]⁻¹ = 405 kip/in.
```

**Step 2 — 패스너 강성 보정**
```
kf = (Es/(1+β)) × [t1/(slip1) + t2/(slip2)]
  β = -0.69 (시험 기반 고정 매개변수)
  slip_i = 0.0379×t1×Pss/(Fu1×ti×d2)
  Pss = 패스너 전단 강도 (lb)
  d2 = 나사 공칭 직경 (in.)
```

**Step 3 — 최종 횡강성**
```
α = kx_plate × fastener_spacing
kx = α × (β × kf)  → 단위: lb/in.
kx_per_unit = kx / fastener_spacing  → 단위: kip/in./in.
```

**Example II-1C 결과:** kx = 1.28 kip/in./in.

### 16.3 왜곡좌굴 매개변수 kφfc, kφwc, kφfg, kφwg 자동 계산

#### 16.3.1 핵심 공식 (Appendix 2, Eq. 2.3.1.3-2)

왜곡좌굴 임계응력의 **핵심 공식**:

```
Fcrd = β × (kφfe + kφwe + kφ) / (k̃φfg + k̃φwg)

여기서:
  kφfe = 플랜지가 웹-플랜지 접합부에 제공하는 탄성 회전강성
  kφwe = 웹이 웹-플랜지 접합부에 제공하는 탄성 회전강성  
  kφ   = 외부 구속(패널/브레이싱)이 제공하는 회전강성
  k̃φfg = 플랜지가 접합부에서 요구하는 기하학적 회전강성
  k̃φwg = 웹이 접합부에서 요구하는 기하학적 회전강성
  β    = 모멘트 구배 보정계수
```

#### 16.3.2 플랜지 기하 성질 계산 (Table 2.3.1.3-1)

C-단면과 Z-단면에 대해 **중심선 치수** b, d, t, θ로부터 계산:

```python
def calc_flange_properties(b, d, t, theta=90.0, section_type='C'):
    """
    Table 2.3.1.3-1: 플랜지+립 기하 성질
    b = 중심선 플랜지폭 = bo - t
    d = 중심선 립높이 = do - t/2
    t = 두께
    theta = 립 각도 (C: 90°, Z: variable)
    """
    import math
    cos_th = math.cos(math.radians(theta))
    sin_th = math.sin(math.radians(theta))
    
    Af = (b + d) * t
    Jf = (b * t**3 + d * t**3) / 3.0
    Cwf = 0.0  # C,Z 단면 모두 0
    
    if section_type == 'C':
        Ixf = t * (t**2 * b**2 + 4*b*d**3 + t**2*b*d + d**4) / (12*(b+d))
        Iyf = t * (b**4 + 4*d*b**3) / (12*(b+d))
        Ixyf = t * b**2 * d**2 / (4*(b+d))
        xof = b**2 / (2*(b+d))
        hxf = -(b**2 + 2*d*b) / (2*(b+d))
        yof = -d**2 / (2*(b+d))
    else:  # Z-section
        Ixf = t*(t**2*b**2 + 4*b*d**3 - 4*b*d**3*cos_th**2 
               + t**2*b*d + d**4 - d**4*cos_th**2) / (12*(b+d))
        Iyf = t*(b**4 + 4*d*b**3 + 6*d**2*b**2*cos_th 
               + 4*d**3*b*cos_th**2 + d**4*cos_th**2) / (12*(b+d))
        Ixyf = t*b*d**2*sin_th*(b + d*cos_th) / (4*(b+d))
        xof = (b**2 - d**2*cos_th) / (2*(b+d))
        hxf = -(b**2 + 2*d*b + d**2*cos_th) / (2*(b+d))
        yof = -d**2*sin_th / (2*(b+d))
    
    return {
        'Af': Af, 'Jf': Jf, 'Cwf': Cwf,
        'Ixf': Ixf, 'Iyf': Iyf, 'Ixyf': Ixyf,
        'xof': xof, 'hxf': hxf, 'yof': yof
    }
```

#### 16.3.3 왜곡좌굴 반파장 Lcrd (Eq. 2.3.1.3-7)

```python
def calc_Lcrd(ho, t, mu, fp):
    """fp = flange properties dict"""
    import math
    term1 = fp['Ixf'] * (fp['xof'] - fp['hxf'])**2 + fp['Cwf']
    term2 = fp['Ixyf']**2 / fp['Iyf'] * (fp['xof'] - fp['hxf'])**2
    
    Lcrd = (6 * math.pi**4 * ho * (1 - mu**2) / t**3 * (term1 - term2))**0.25
    return Lcrd
```

#### 16.3.4 4개 강성항 계산 (Eq. 2.3.1.3-3 ~ 2.3.3.3-6)

```python
def calc_distortional_stiffness(ho, t, mu, E, G, fp, L, xi_web=0):
    """
    ho: 웹 외-외 깊이
    xi_web: 응력 구배 (휨: 0=균일압축, 2=순수휨)
    """
    import math
    pi = math.pi
    piL = pi / L
    
    # 탄성 회전강성 — 플랜지 (Eq. 2.3.1.3-3)
    term_a = E * fp['Ixf'] * (fp['xof'] - fp['hxf'])**2 + E * fp['Cwf']
    term_b = E * fp['Ixyf']**2 / fp['Iyf'] * (fp['xof'] - fp['hxf'])**2
    k_phi_fe = piL**4 * (term_a - term_b) + piL**2 * G * fp['Jf']
    
    # 탄성 회전강성 — 웹 (압축 부재: Eq. 2.3.1.3-4, 휨 부재: Eq. 2.3.3.3-5)
    if xi_web == 0:  # 압축
        k_phi_we = E * t**3 / (6 * ho * (1 - mu**2))
    else:  # 휨
        k_phi_we = (E * t**3 / (12 * (1 - mu**2))) * (
            3/ho + piL**2 * 19*ho/60 + piL**4 * ho**3/240
        )
    
    # 기하학적 회전강성 — 플랜지 (Eq. 2.3.1.3-5)
    A = fp['Af']
    r1 = (fp['xof'] - fp['hxf'])**2 * (fp['Ixyf']/fp['Iyf'])**2
    r2 = 2 * fp['yof'] * (fp['xof'] - fp['hxf']) * fp['Ixyf']/fp['Iyf']
    r3 = fp['hxf']**2 + fp['yof']**2
    k_tilde_phi_fg = piL**2 * (A * (r1 - r2 + r3) + fp['Ixf'] + fp['Iyf'])
    
    # 기하학적 회전강성 — 웹 (압축: Eq. 2.3.1.3-6, 휨: Eq. 2.3.3.3-6)
    if xi_web == 0:  # 압축
        k_tilde_phi_wg = piL**2 * t * ho**3 / 60
    else:  # 휨 (xi_web=2 for pure bending)
        LH = L / ho
        HL = ho / L
        num = ((45360*(1-xi_web)+62160)*LH**2 + 448*pi**2 
               + HL**2*(53+3*(1-xi_web))*pi**4)
        den = pi**4 + 28*pi**2*LH**2 + 420*LH**4
        k_tilde_phi_wg = ho*t*pi**2/13440 * (num/den)
    
    return {
        'k_phi_fe': k_phi_fe,  # kip
        'k_phi_we': k_phi_we,  # kip
        'k_tilde_phi_fg': k_tilde_phi_fg,  # in.²
        'k_tilde_phi_wg': k_tilde_phi_wg   # in.²
    }
```

### 16.4 비틀림 하중 및 바이모멘트 계산

#### 16.4.1 하중 편심에 의한 비틀림 모멘트

C-단면/Z-단면에 하중이 웹 면내로 작용하면, 전단중심(Shear Center)을 지나지 않아 비틀림이 발생:

```
편심 e = m + t/2 + R
  m = 전단중심~웹 중심선 거리 (Table I-1 또는 get_cutwp에서 제공)
  t = 플랜지 두께
  R = 코너 반경

단위길이당 비틀림 모멘트:
  tu = wu × e  (kip-in./ft)
```

#### 16.4.2 비틀림 해석 — AISC DG9 방법

**핵심 매개변수:**
```
a = √(E×Cw / G×J)    ← 비틀림 특성 길이
  E = 29500 ksi, G = 11300 ksi
  Cw = 뒤틀림 상수 (in.⁶) ← get_cutwp 제공
  J = 생로랑 비틀림 상수 (in.⁴)
```

**DG9 Case 4: 등분포 비틀림 하중, 양단 핀**
```
θ″(z) = (tr/GJ) × [-1 + cosh(z/a) - tanh(L/2a)×sinh(z/a)]

여기서:
  tr = tu/12  (kip-in./in.으로 변환)
  z = 부재 내 위치 (in.)
  L = 스팬 (in.)
```

**DG9 Case 3: 집중 비틀림 하중 (브레이스 반력)**
```
θ″T(z) = (T/aGJ) × [sinh(αL/a)×cosh(z/a) - cosh(αL/a)×sinh(z/a)] / tanh(L/a)
  T = 브레이스 반력 (superposition으로 결정)
  α = 브레이스 위치비 (예: 0.5 = 중앙)
```

#### 16.4.3 워핑 응력 계산

**워핑 수직응력:**
```
f_torsion = E × Wno × θ″     (AISC DG9 Eq. 4.3a)
         = B × Wno / Cw       (바이모멘트 사용 시)

여기서:
  Wno = 정규화 워핑 함수 (단면의 각 점에서 다름)
  B = E × Cw × θ″  (바이모멘트, kip-in.²)
```

**C-단면의 워핑 함수 (3개 핵심점):**
```
Point A (립 끝):  WA = ā(m-b̄)/2 - c̄(m+b̄)
Point B (플랜지-립 접합): WB = ā(m-b̄)/2
Point C (플랜지-웹 접합): WC = ā×m/2

여기서:
  ā = 중심선 웹높이
  b̄ = 중심선 플랜지폭
  c̄ = 중심선 립길이
  m = 전단중심~웹 중심선 거리
```

#### 16.4.4 비틀림 감소계수 R (Section H4)

```
R = f_bending_max / (f_bending + f_torsion) ≤ 1.0

여기서:
  f_bending_max = Mu/Sf (극단 파이버에서의 휨응력)
  f_bending = 검토점에서의 휨응력 (= Mu×c/Ix)
  f_torsion = 검토점에서의 워핑 수직응력

Example II-1C 결과:
  R = 34.3 / (33.9 + 1.70) = 0.963
  φRMnℓ = 0.9 × 0.963 × 93.6 = 81.1 kip-in. > 78.5 → OK
```

#### 16.4.5 코드 구현 — 비틀림 해석 단순화

MASTAN2처럼 완전한 프레임 해석 대신, **단순보에 대한 DG9 해석해**를 구현:

```python
def calc_torsion_reduction_R(
    section_props: dict,  # Ix, Sf, J, Cw, m, xo
    span_ft: float,
    wu_plf: float,       # 계수하중 (LRFD)
    brace_locations: list,  # [0.5] = 중앙, [0.33, 0.67] = 1/3점
    E: float = 29500, G: float = 11300
) -> float:
    """
    AISC DG9 기반 비틀림 감소계수 R 계산
    
    Returns: R (0~1.0)
    """
    import math
    
    L = span_ft * 12  # in.
    e = section_props['m'] + section_props.get('t', 0.059)/2 + section_props.get('R', 0.1875)
    tr = wu_plf * e / (12 * 1000)  # kip-in./in.
    
    a = math.sqrt(E * section_props['Cw'] / (G * section_props['J']))
    
    # DG9 Case 4: θ″ at midspan (z = L/2)
    z = L / 2
    theta_pp_dist = (tr / (G * section_props['J'])) * (
        -1.0 + math.cosh(z/a) - math.tanh(L/(2*a)) * math.sinh(z/a)
    )
    
    # 브레이스 반력은 superposition으로 θ=0 조건에서 결정
    # ... (브레이스 수에 따라 DG9 Case 3 중첩)
    
    # 워핑 응력 계산
    Wno_max = max(abs(w) for w in section_props['Wno_values'])
    f_torsion = abs(E * Wno_max * theta_pp_dist)
    
    # 휨응력
    f_bending_max = wu_plf * L**2 / (8 * 1000) / section_props['Sf']  # ksi
    
    # R 계산
    R = f_bending_max / (f_bending_max + f_torsion)
    return min(R, 1.0)
```

### 16.6 모멘트 구배 보정계수 상세

#### 16.6.1 Cb — 횡-비틀림좌굴 보정 (Eq. F2.1.1-2)

```
Cb = 12.5 × Mmax / (2.5×Mmax + 3×MA + 4×MB + 3×MC)

여기서 (비지지 구간 내):
  Mmax = 구간 내 최대 모멘트 절대값
  MA = 1/4점 모멘트 절대값
  MB = 중앙점 모멘트 절대값
  MC = 3/4점 모멘트 절대값

보수적으로 Cb = 1.0 항상 허용
캔틸레버(자유단 비지지): Cb = 1.0 강제
```

**예제에서의 사용:** 부모멘트 영역에서 보수적으로 Cb = 1.67 (선형 가정)

**Fcre 공식 — 단면 유형별 차이 (★ 중요):**

| 단면 유형 | Fcre 공식 | 핵심 차이 |
|----------|----------|----------|
| C-단면 (대칭축 휨) | Fcre = Cb×ro×A/Sf × √(σey×σt) (Eq. F2.1.1-1) | 표준 |
| C-단면 (약축 휨) | Fcre = Cs×A×σex/(CTF×Sf) × [j+Cs√(j²+ro²σt/σex)] (Eq. F2.1.2-1) | j, Cs, CTF 추가 |
| Z-단면 (점대칭) | Fcre = Cb×ro×A/**2**Sf × √(σey×σt) (Eq. F2.1.3-1) | **분모에 2** (절반 강도!) |
| 간략식 (C-단면) | Fcre = Cb×π²EdIyc / (Sf(KyLy)²) (Eq. F2.1.1-6) | 이중 대칭 I형과 동일 |
| 간략식 (Z-단면) | Fcre = Cb×π²EdIyc / (**2**Sf(KyLy)²) (Eq. F2.1.3-2) | **분모에 2** |

**Fn 결정 (3단계 커브):**
```
Fcre ≥ 2.78Fy  →  Fn = Fy                              (항복 지배)
0.56Fy < Fcre < 2.78Fy  →  Fn = 10/9×Fy×(1-10Fy/36Fcre)  (비탄성)
Fcre ≤ 0.56Fy  →  Fn = Fcre                             (탄성좌굴)
```

```python
def calc_Fcre_and_Fn(section_type, Cb, E, G, Sf, A, ro, Iy, Iyc,
                     J, Cw, d, Fy, Ky, Ly, Kt, Lt, ry):
    """Section F2.1 — Fcre 및 Fn 자동 계산"""
    import math
    
    sigma_ey = math.pi**2 * E / (Ky * Ly / ry)**2
    sigma_t = (1 / (A * ro**2)) * (G * J + math.pi**2 * E * Cw / (Kt * Lt)**2)
    
    if section_type == 'Z':
        Fcre = Cb * ro * A / (2 * Sf) * math.sqrt(sigma_ey * sigma_t)
    else:  # C or doubly-symmetric
        Fcre = Cb * ro * A / Sf * math.sqrt(sigma_ey * sigma_t)
    
    # Fn 결정 (3단계)
    if Fcre >= 2.78 * Fy:
        Fn = Fy
    elif Fcre > 0.56 * Fy:
        Fn = 10/9 * Fy * (1 - 10*Fy / (36*Fcre))
    else:
        Fn = Fcre
    
    Mne = Sf * Fn
    return {'Fcre': Fcre, 'Fn': Fn, 'Mne': Mne, 'sigma_ey': sigma_ey, 'sigma_t': sigma_t}
```

#### 16.6.2 β — 왜곡좌굴 구배 보정 (Appendix 2, Eq. 2.3.3.3-3)

```
β = 1 + 0.4 × (L/Lm)^0.7 × (1 + M1/M2)^0.7

제한: 1.0 ≤ β ≤ 1.3

여기서:
  L = min(Lcrd, Lm)
  Lm = 왜곡좌굴 비지지 길이 (랩 끝 ~ 변곡점)
  M1 = 구간 시작점 모멘트 (변곡점에서 보통 0)
  M2 = 구간 끝점 모멘트 (랩 끝에서의 모멘트)
```

**자동 계산 구현:**
```python
def calc_beta_distortional(Lcrd, Lm, M1, M2):
    """Appendix 2, Eq. 2.3.3.3-3"""
    L = min(Lcrd, Lm)
    if M2 == 0:
        return 1.0
    beta = 1.0 + 0.4 * (L/Lm)**0.7 * (1.0 + abs(M1/M2))**0.7
    return max(1.0, min(beta, 1.3))
```

```python
def select_web_crippling_case(location, num_flanges, fastened):

    case = f"{'E' if location=='end' else 'I'}{'O' if num_flanges==1 else 'T'}F"
    table = 'fastened' if fastened else 'unfastened'
    return case, table  # 예: ('EOF', 'fastened') → Table G5-1
```

### 16.8 연속보 구조해석 구현 방법

#### 16.8.1 3-모멘트법 (Three-Moment Equation, Clapeyron)

등단면 등경간 연속보에 대한 해석적 해법. 구현이 간단하고 정확:

```
Mi-1×Li + 2×Mi×(Li + Li+1) + Mi+1×Li+1 = -6×(Ai×āi/Li + Ai+1×b̄i+1/Li+1)

여기서:
  Mi = i번째 지점 모멘트
  Li = i번째 스팬 길이
  Ai = i번째 스팬의 하중에 의한 단순보 모멘트 면적
  āi, b̄i = 모멘트 면적의 도심 위치
```

#### 16.8.2 비등단면 연속보 — 강성행렬법

랩 구간에서 Ix가 변하는 비등단면에는 **직접 강성법** 사용:

```javascript
    setValue(plfId, (psf * spacing).toFixed(0));
}

// Calculator 모드 판별
function isCalculatorMode(memberApp) {
    return ['roof-purlin','floor-joist','wall-girt','wall-stud'].includes(memberApp);
}

// Analyze Loads 결과 렌더링
function renderLoadAnalysisResult(data) {
    const el = document.getElementById('load-analysis-result');
    if (!el || !data) return;
    
    // M/V 다이어그램 (Canvas 또는 SVG)
    renderMomentDiagram(data.gravity.M_diagram, 'moment-diagram-canvas');
    renderShearDiagram(data.gravity.V_diagram, 'shear-diagram-canvas');
    
    // 임계 위치 테이블
    let html = '<table class="result-table"><tr><th>Location</th><th>Mu(k-ft)</th><th>Vu(kips)</th><th>Ru(kips)</th></tr>';
    for (const loc of data.gravity.locations) {
        html += `<tr><td>${loc.name}</td><td>${loc.Mu?.toFixed(2)||'—'}</td>
                 <td>${loc.Vu?.toFixed(2)||'—'}</td><td>${loc.Ru?.toFixed(2)||'—'}</td></tr>`;
    }
    html += '</table>';
    
    // Auto params
    html += `<div class="auto-params">
        <p>Positive: Fully braced, kφ=${data.auto_params.deck.kphi.toFixed(3)}</p>
        <p>Negative: Ly=${data.auto_params.negative_region.Ly.toFixed(1)}in, Cb=${data.auto_params.negative_region.Cb}</p>
        <p>Uplift R=${data.auto_params.uplift_R}</p>
    </div>`;
    
    el.innerHTML = html;
    
    // Required Loads 필드 자동 채우기 (지배 위치 기준)
    // 사용자가 수정 가능하도록 input에 값 설정
    const gov = data.gravity.locations.reduce((max, l) => 
        Math.abs(l.Mu||0) > Math.abs(max.Mu||0) ? l : max);
    setValue('design-Mx', (Math.abs(gov.Mu) * 12).toFixed(1)); // kip-ft → kip-in
    setValue('design-V', (gov.Vu || 0).toFixed(2));
}
```

## 4. CUFSM Modeling Technique for Perforations

### Net Section Modeling in CUFSM

1. Start with gross section geometry (nodes and elements)
2. At perforation locations: **set element thickness = 0** (do NOT delete elements)
3. Set stress state to "nominal" to obtain net section properties
4. This simulates the reduced stiffness at the hole location

### Distortional Buckling with Hole Effect

Reduced web thickness for distortional analysis:

tr = t * (1 - Lh/Lcrd)^(1/3) = 0.0346 * (1 - 4.5/16.6)^(1/3) = 0.0311 in.

| Parameter | Without holes | With holes | Reduction |
|-----------|--------------|------------|-----------|
| Mcrd | 23.4 kip-in. | 20.5 kip-in. | -12% |

## 8. Key Features for Validation

- **Perforation modeling technique**: Element thickness = 0 at holes (critical CUFSM technique)
- **Reduced thickness method**: tr = t*(1 - Lh/Lcrd)^(1/3) for distortional buckling with holes
- **Two CUFSM analyses**: Gross section (buckling curve) + net section (t=0 at hole)
- **Section property calculator**: CUFSM also used for computing gross/net section properties
- **My vs My,net distinction**: Important for perforation design
- **Hole effect quantified**: 12% reduction in Mcrd, 15% reduction in overall strength

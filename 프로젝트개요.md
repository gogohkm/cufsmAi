# CUFSM 프로젝트 개요

## 1. 프로젝트 소개

**CUFSM** (Constrained and Unconstrained Finite Strip Method)은 MATLAB으로 작성된 냉간성형강(Cold-Formed Steel) 부재의 단면 탄성 좌굴 해석 프로그램이다. 유한스트립법(FSM)을 기반으로 하며, 구조공학자와 연구자들이 박판 단면의 좌굴 거동을 분석하는 데 사용된다.

- **현재 버전:** 5.68 (2025년 12월 2일)
- **라이선스:** MIT
- **원본 저장소:** https://github.com/thinwalled/cufsm-git
- **주요 개발자:**
  - Benjamin W. Schafer (Johns Hopkins University) - 총괄 개발
  - Sandor Adany (Budapest University of Technology) - cFSM/GBT 이론
  - Zhanjie Li - 일반 경계조건 구현, 벡터화
  - Sheng Jin - 3D 시각화, fcFSM, 진동 해석

---

## 2. 핵심 기능

| 기능 | 설명 |
|------|------|
| **탄성 좌굴 해석 (FSM)** | 유한스트립법 기반 고유치 문제 풀이: [K - λ·Kg]{Φ} = 0 |
| **구속 유한스트립법 (cFSM)** | GBT 기반 모드 분류 (전체/뒤틀림/국부/기타) |
| **힘 기반 구속 유한스트립법 (fcFSM)** | 힘 평형 기반 모드 분해 (배치 모드 전용) |
| **일반 경계조건** | S-S, C-C, S-C, C-F, C-G 5가지 단부조건 지원 |
| **자유 진동 해석** | 고유진동수 및 모드형상 계산 (배치 모드 전용) |
| **소성 상호작용곡면** | P-Mxx-Mzz 소성 상호작용 표면 생성 (DSM 설계용) |
| **단면 템플릿** | C형, Z형, Hat형, HDS, 원형관 등 파라메트릭 단면 생성 |
| **스프링 요소** | 분포/집중 스프링 (축방향, 횡방향, 수직, 비틀림 DOF) |
| **벡터화 연산** | 대규모 해석을 위한 성능 최적화 (v5.20+) |
| **CUTWP 연동** | 뒤틀림 단면 성질 계산 (비틀림 상수 Cw 등) |

## 4. 해석 워크플로우

```
[1] 단면 정의          [2] 재료 물성          [3] 하중/경계조건
 (pre2.m, template)     (E, v, G, rho)        (P, Mxx, Mzz, BC)
       │                      │                      │
       └──────────┬───────────┘──────────────────────┘
                  ▼
        [4] 해석 설정 (선택)
         ├─ cFSM 모드 선택 (G/D/L/O)
         ├─ 스프링 조건
         └─ 구속 방정식
                  │
                  ▼
        [5] 해석 실행 (stripmain.m)
         ├─ 요소별 klocal, kglocal 계산
         ├─ 전체 행렬 K, Kg 조립
         ├─ 스프링/구속 적용
         ├─ 경계조건 적용 (형상함수)
         └─ 일반화 고유치 문제 풀이
                  │
                  ▼
        [6] 후처리 및 시각화
         ├─ 좌굴 곡선 (하중비 vs 길이)
         ├─ 2D/3D 모드형상 표시
         ├─ 모드 분류 (cFSM 사용 시)
         └─ 다중 해석 결과 비교
```

**핵심 고유치 문제:**
- **좌굴 해석:** `[K - λ·Kg]{Φ} = 0` (λ = 좌굴 하중비, Φ = 좌굴 모드형상)
- **진동 해석:** `[K - omega^2·M]{Φ} = 0` (omega = 고유진동수, M = 질량행렬)

## 5. 주요 모듈 상세

### 5.1 FSM 해석 엔진 (`analysis/`)

유한스트립법의 핵심 구현부로, 박판 스트립 요소를 사용하여 단면의 좌굴 거동을 계산한다.

- **stripmain.m**: 전체 해석 흐름을 제어하는 메인 루틴. 각 길이/종방향 항(m)에 대해 요소 행렬을 조립하고 고유치 문제를 풀이한다.
- **klocal.m / kglocal.m**: 개별 스트립 요소의 탄성 강성행렬과 기하 강성행렬(P-delta 효과)을 구성한다.
- **BC_I1_5.m**: 5가지 경계조건(S-S, C-C, S-C, C-F, C-G)에 대한 종방향 형상함수의 적분값을 계산한다. 이는 일반 경계조건 처리의 핵심이다.

### 5.2 구속 유한스트립법 (`analysis/cFSM/`)

일반화 보 이론(GBT) 기반 기저벡터를 사용하여 좌굴 모드를 물리적 의미에 따라 분류한다:
- **전체(G, Global):** 부재 전체의 휨/비틀림
- **뒤틀림(D, Distortional):** 단면 벽체의 뒤틀림 변형
- **국부(L, Local):** 개별 판의 국부 좌굴
- **기타(O, Other):** 나머지 변형 모드

### 5.3 힘 기반 구속 FSM (`analysis/fcFSM/`)

기존 cFSM의 운동학적 접근 대신 힘 평형을 기반으로 모드를 분해하는 대안적 방법. 곡선 모서리를 가진 단면도 지원하며, 현재 배치 모드로만 사용 가능하다.

### 5.4 소성 해석 (`analysis/plastic/`)

파이버 기반 단면 이산화를 통해 P-Mxx-Mzz 소성 상호작용 곡면을 생성한다. AISI DSM(Direct Strength Method) 설계에 필요한 beta, theta, phi 파라미터를 산출한다.

### 5.5 자유 진동 해석 (`analysis/vibration/`)

v5.60에서 추가된 기능으로, 질량행렬을 포함한 고유치 문제를 풀어 고유진동수와 모드형상을 계산한다. 재료 밀도(rho) 지정이 필요하며, 현재 배치 모드 전용이다.

## 6. 경계조건 지원

| 코드 | 경계조건 | 설명 |
|------|---------|------|
| S-S | Simply-Supported / Simply-Supported | 양단 단순지지 (시그니처 곡선) |
| C-C | Clamped / Clamped | 양단 고정 |
| S-C | Simply-Supported / Clamped | 한쪽 단순지지, 한쪽 고정 |
| C-F | Clamped / Free | 한쪽 고정, 한쪽 자유 |
| C-G | Clamped / Guided | 한쪽 고정, 한쪽 안내 |
